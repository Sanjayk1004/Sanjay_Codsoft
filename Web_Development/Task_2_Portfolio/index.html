Electricity Billing System

The proposed electricity billing system aims to provide an efficient and user-friendly solution for both the admin and users. In this system, the admin will be able to add customer data, while users can conveniently view their bills. This idea is implemented by using frontend as Reactjs and backend as SpringBoot.
